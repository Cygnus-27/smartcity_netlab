# SmartCity NetLab

An interactive 2D sandbox for simulating network routing protocols (OSPF and RIP) within modern smart city infrastructures. 

### Key Features
* **Custom Topologies**: Drag-and-drop nodes to create and modify networks in real-time.
* **Step-by-Step Simulation**: Granular playback controls with adjustable speeds and per-node logic captions.
* **Algorithm Comparison**: Compare Link-State (OSPF) vs. Distance Vector (RIP) convergence behaviors.
* **Real-time Metrics**: Track convergence time and control message overhead dynamically.

### Setup
```bash
npm install
npm start
```
